# Tatar Learn Frontend

Telegram Mini App для изучения татарского языка.

## Деплой на Vercel

1. Создай репозиторий на GitHub
2. Загрузи `index.html`
3. Подключи к Vercel
4. В настройках измени `API_URL` на свой домен Amvera

## Настройка

В файле `index.html` найди строку:
```javascript
const API_URL = '...';
```

Замени `https://tatar-backend.amvera.io` на URL твоего бэкенда на Amvera.

## Настройка бота

1. Создай бота через @BotFather
2. `/mybots` → выбери бота → Bot Settings → Menu Button → Configure
3. Укажи URL от Vercel

## Функционал

- 10 бесплатных уроков
- 40 премиум уроков (лексика + грамматика)
- Оплата через Telegram Stars
- Сохранение прогресса
- Система XP и streak
